Chặn Windows Feature Updates bằng Registry Editor

Hướng dẫn cách sử dụng Registry Editor để chặn Windows tự động cập nhật lên các phiên bản feature update mới, giúp bạn kiểm soát tốt hơn quá trình cập nhật hệ thống.

Windows thường tự động cài đặt các feature update mới, điều này có thể gây gián đoạn công việc hoặc đưa ra những thay đổi không mong muốn. Nếu bạn muốn giữ nguyên phiên bản Windows hiện tại (ví dụ: Windows 11 23H2) và chỉ nhận các bản cập nhật bảo mật, bạn có thể sử dụng Registry Editor để chặn feature updates. Bài viết này sẽ hướng dẫn bạn cách thực hiện điều đó một cách an toàn.

Tại sao tôi chặn Feature Updates?

Feature Updates là những bản cập nhật lớn của Windows, thường mang đến các tính năng mới và thay đổi đáng kể về giao diện. Tuy nhiên, không phải lúc nào việc cập nhật ngay lập tức cũng là điều tốt. Trong trường hợp của tôi, sau khi Windows tự động cập nhật lên phiên bản 24H2, Steam của tôi bắt đầu gặp lỗi và không hoạt động bình thường. Điều này gây ảnh hưởng đáng kể đến trải nghiệm chơi game của tôi.

Vì vậy, tôi quyết định chặn feature updates để giữ nguyên phiên bản Windows 23H2 đang hoạt động ổn định. Bằng cách này, tôi vẫn sẽ nhận được các bản cập nhật bảo mật quan trọng nhưng sẽ không bị "ép" nâng cấp lên phiên bản mới có thể gây ra các vấn đề tương thích với phần mềm mà tôi đang sử dụng. Đây là một giải pháp hữu ích khi bạn muốn kiểm soát tốt hơn quá trình cập nhật hệ thống và tránh những rủi ro không mong muốn.

Lưu ý quan trọng: Trước khi thực hiện bất kỳ thay đổi nào trong Registry Editor, hãy đảm bảo bạn đã sao lưu registry hoặc tạo System Restore Point. Registry Editor là công cụ mạnh mẽ và việc chỉnh sửa sai có thể gây ảnh hưởng đến hệ thống.

Hướng dẫn từng bước

Bước 1: Mở Registry Editor

Đầu tiên, bạn cần mở Registry Editor (regedit). Có nhiều cách để làm điều này:

  • Cách 1: Nhấn tổ hợp phím Win + R, gõ regedit và nhấn Enter
  • Cách 2: Tìm kiếm "Registry Editor" trong Start Menu và chọn kết quả tương ứng
  • Cách 3: Mở Command Prompt hoặc PowerShell với quyền Administrator, gõ regedit và nhấn Enter

Khi Registry Editor mở ra, bạn sẽ thấy một cây thư mục bên trái với các key registry. Đây là nơi bạn sẽ thực hiện các thay đổi.

Bước 2: Điều hướng đến key WindowsUpdate

Trong Registry Editor, bạn cần điều hướng đến đường dẫn sau: 

HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate

Để làm điều này, hãy mở rộng từng thư mục theo thứ tự:

  • Mở HKEY_LOCAL_MACHINE
  • Mở SOFTWARE
  • Mở Policies
  • Mở Microsoft
  • Mở Windows
  • Kiểm tra xem có key WindowsUpdate hay chưa

Nếu key WindowsUpdate chưa tồn tại: Bạn cần tạo key mới. Nhấp chuột phải vào key Windows, chọn New → Key, và đặt tên là WindowsUpdate.

Nếu key WindowsUpdate đã tồn tại: Bạn có thể tiếp tục với bước tiếp theo.

Bước 3: Tạo các giá trị Registry

Sau khi đã điều hướng đến key WindowsUpdate, bạn cần tạo hai giá trị registry sau:

Tạo DWORD TargetReleaseVersion

Đầu tiên, tạo một giá trị DWORD (32-bit):

  • Nhấp chuột phải vào vùng trống bên phải trong key WindowsUpdate
  • Chọn New → DWORD (32-bit) Value
  • Đặt tên là: TargetReleaseVersion
  • Nhấp đúp vào giá trị vừa tạo để mở hộp thoại Edit
  • Trong trường Value data, nhập: 1
  • Đảm bảo Base được chọn là Hexadecimal hoặc Decimal (cả hai đều hoạt động với giá trị 1)
  • Nhấn OK để lưu

Tạo String TargetReleaseVersionInfo

Tiếp theo, tạo một giá trị String (REG_SZ):

  • Nhấp chuột phải vào vùng trống bên phải trong key WindowsUpdate
  • Chọn New → String Value
  • Đặt tên là: TargetReleaseVersionInfo
  • Nhấp đúp vào giá trị vừa tạo để mở hộp thoại Edit
  • Trong trường Value data, nhập: 23H2
  • Nhấn OK để lưu

Lưu ý: Giá trị 23H2 tương ứng với phiên bản Windows 11 23H2. Nếu bạn đang sử dụng phiên bản khác (ví dụ: 22H2, 21H2), hãy thay đổi giá trị này cho phù hợp. Bạn có thể kiểm tra phiên bản Windows hiện tại bằng cách mở Settings → System → About.

Bước 4: Khởi động lại máy

Sau khi đã tạo xong cả hai giá trị registry, bạn cần khởi động lại máy tính để các thay đổi có hiệu lực. Đóng Registry Editor và khởi động lại Windows.

Sau khi khởi động lại, Windows sẽ không tự động cài đặt các feature update mới hơn phiên bản bạn đã chỉ định (23H2 trong trường hợp này). Hệ thống vẫn sẽ nhận và cài đặt các bản cập nhật bảo mật quan trọng, nhưng sẽ bỏ qua các feature update.

Kiểm tra kết quả: Sau khi khởi động lại, bạn có thể kiểm tra xem cài đặt đã hoạt động chưa bằng cách mở Settings → Windows Update. Nếu cài đặt thành công, bạn sẽ thấy thông báo rằng Windows đang ở phiên bản được chỉ định và sẽ không tự động nâng cấp lên phiên bản mới hơn.

Cách hoàn nguyên thay đổi

Nếu sau này bạn muốn cho phép Windows tự động cập nhật feature updates trở lại, bạn có thể:

  • Xóa hai giá trị TargetReleaseVersion và TargetReleaseVersionInfo trong Registry Editor
  • Hoặc đơn giản là xóa toàn bộ key WindowsUpdate nếu bạn không cần các cài đặt khác trong key này
  • Sau đó khởi động lại máy để áp dụng thay đổi

Kết luận

Bằng cách sử dụng Registry Editor để thiết lập TargetReleaseVersion và TargetReleaseVersionInfo, bạn có thể kiểm soát tốt hơn quá trình cập nhật Windows. Phương pháp này đặc biệt hữu ích khi bạn muốn giữ nguyên phiên bản Windows hiện tại để đảm bảo tính ổn định và tương thích với các phần mềm quan trọng. Hãy nhớ luôn sao lưu registry trước khi thực hiện bất kỳ thay đổi nào và chỉ chỉnh sửa khi bạn hiểu rõ những gì mình đang làm.
