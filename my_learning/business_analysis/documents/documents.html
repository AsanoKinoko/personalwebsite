Các loại tài liệu (Documents)

BA luôn xoay quanh 4 thứ:

  • Hiểu business -> Business Case
  • Define requirement -> BRD / SRS / User Story
  • Mô hình hóa -> Diagram / Flow
  • Đảm bảo đúng -> Test / UAT / RTM

Không phải dự án nào cũng dùng hết, nhưng core documents BA gần như luôn có:

  • BRD / SRS / User Story
  • Use Case / Flow
  • Wireframe
  • RTM
  • Test Case / UAT

1. BRD - Business Requirements Document

  • Mô tả yêu cầu ở mức business
  • Tập trung vào "WHAT & WHY"

2. FRD - Functional Requirements Document

  • Mô tả chức năng cụ thể
  • Flow xử lý, logic

3. NFR - Non-functional Requirements

  • Performance
  • Security
  • Scalability
  • Availability
  • Usability

4. SRS - Software Requirements Specification

Chi tiết hóa yêu cầu hệ thống

Tập trung vào "WHAT system should do"

  • User Stories (Agile):
    • Format: As a ... I want ... so that ...
    • Acceptance Criteria (AC)
  • Use Case Specification:
    • Actors
    • Main flow / Alternative flow

5. Tài liệu kiểm thử (Testing)

BA thường hỗ trợ QA

  • Test Plan
  • Test Case
  • Test Scenario
  • UAT Document (User Acceptance Test)
  • Acceptance Criteria (AC)
  • Traceability Matrix (RTM)

6. Tài liệu kỹ thuật hỗ trợ Dev

BA thường bridge sang dev

  • API Specification (Swagger / OpenAPI)
  • Integration Document
  • System Architecture Overview
  • Database Schema
  • Business Rules Document
  • Mapping Document (Data mapping)

7. Tài liệu quản lý yêu cầu (Requirement Management)

  • Requirement Traceability Matrix (RTM)
  • Backlog (Product Backlog / Sprint Backlog)
  • Prioritization Document (MoSCoW, RICE...)
  • Requirement Catalogue

8. Tài liệu UX/UI & Product

Dùng cho thiết kế giao diện & trải nghiệm

  • Wireframe
  • Mockup
  • Prototype (Figma, Adobe XD)
  • User Flow
  • Customer Journey Map
  • Personas

9. Tài liệu triển khai & vận hành

  • Release Notes
  • Deployment Guide
  • User Manual / User Guide
  • Training Materials
  • SOP (Standard Operating Procedure)
  • Support Document

10. Một số tài liệu phân tích nâng cao (Advanced BA) có thể có

  • Gap Analysis
  • Root Cause Analysis (RCA)
  • SWOT Analysis
  • PESTLE Analysis
  • Risk Analysis Document
  • KPI / Metrics Definition
