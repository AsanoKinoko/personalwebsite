Làm việc với component

Component là một UI element có thể tái sử dụng nhiều lần.

Ví dụ:

  • Button
  • Card
  • Navbar
  • Input
  • Modal

Ý chính: Bạn tạo 1 lần → dùng lại ở nhiều nơi → sửa 1 chỗ → update toàn bộ.

Tạo component

Để tạo một component thì bạn hãy chọn đối tượng, sau đó nhấn vào ở thanh công cụ rồi chọn Create component hoặc bấm tổ hợp Ctrl + Alt + K.
