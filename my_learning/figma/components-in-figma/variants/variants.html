Variants

Đây là kiến thức quan trọng về component. Variants cho phép nhiều trạng thái trong 1 component.

Ví dụ Button:

Variant (State):

  • Default
  • Hover
  • Active
  • Disabled

Hoặc:

Size:

  • Small
  • Medium
  • Large

Thêm variant

Để thêm một Variant thì bạn bấm nút Add variant ở bên cột bên phải:

Sau khi add:

Sau khi tạo bạn có thể sửa đổi Property của Variant vừa tạo ở cột bên phải:

Các loại property

  • Text property: Cho phép đổi text. VD: Button label = "Buy now"
  • Boolean property: Bật/tắt element. VD: Icon visible = true / false
  • Instance swap property: Cho phép đổi component con. VD: Icon = [Cart / Heart / User]
  • Variant property: Chọn variant. VD: Size = Large; State = Hover
  • Slot: dùng để cho phép “nhét” nội dung linh hoạt vào bên trong component. Slot là vùng “placeholder” bên trong component để chứa nội dung tuỳ biến. Hiểu đơn giản: 
    Component (Card)
├─ Image
├─ Title
└─ Slot (content)

    Slot cho phép bạn thay đổi nội dung bên trong mà không phá structure component.
