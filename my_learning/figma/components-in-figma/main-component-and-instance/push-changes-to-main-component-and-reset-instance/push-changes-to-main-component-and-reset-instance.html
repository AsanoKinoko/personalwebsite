Push changes to main component và Reset instance

Push changes to main component: Đẩy thay đổi từ instance lên component gốc

Ví dụ: Bạn có

  • Button (Main): "Submit"
  • Instance: "Login"

Bạn sửa instance:

  • đổi màu
  • đổi padding
  • đổi layout

Bấm Push changes → Main component sẽ trở thành: Button (Main): "Login" + style mới → Tất cả instance khác cũng update theo

Khi nào nên dùng?

Khi:

  • bạn chỉnh instance và thấy nó đúng hơn main
  • muốn update toàn hệ thống

Không nên khi:

  • bạn chỉ test tạm
  • thay đổi chỉ dùng cho 1 instance

Reset instance là xoá tất cả override của instance → trở về giống main component

Ví dụ:

  • Main: Button (Blue, "Submit")
  • Instance: (Red, "Login")

Bấm Reset instance → Instance trở về: Blue, "Submit"
