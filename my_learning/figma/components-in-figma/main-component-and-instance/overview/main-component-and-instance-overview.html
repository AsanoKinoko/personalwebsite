Tổng quan

Để tạo một instance của 1 component, chúng ta có những cách sau:

  • Kéo từ Assets panel (cách chuẩn nhất): Mở tab Assets → Tìm component → Drag & drop vào canvas

    Ưu điểm:

    • Nhanh
    • Dễ quản lý
    • Chuẩn khi làm design system
  • Alt + Drag (duplicate instance): Chọn một instance đã có → giữ Alt + kéo → tạo instance mới
  • Dùng khi: Muốn copy nhanh UI đang có
  • Copy / Paste: Ctrl + C → Ctrl + V → tạo instance mới giống hệt
  • Duplicate (Ctrl + D): Nhân bản instance (nhanh hơn copy/paste)
  • Quick Insert (Ctrl + /): Nhấn Ctrl + / → gõ tên component → Enter → instance được tạo ngay (rất nhanh cho power user)

Best practice (rất quan trọng)

  • Luôn tạo instance từ Assets khi làm project lớn
  • Tránh copy lung tung từ nhiều nơi
  • Giữ structure rõ ràng

Main Component (component gốc)

  • Là “source of truth”
  • Có thể chỉnh sửa full

Instance (bản copy)

  • Là bản sử dụng của component
  • Kế thừa từ main component
Main Button
├─ Instance 1
├─ Instance 2
└─ Instance 3

Khi sửa main → tất cả instance update.

VD: Khi đổi màu tam giác ở Main component thì tam giác ở tất cả instance đổi màu theo.
