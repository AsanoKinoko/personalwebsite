3. Export

Trong Figma, Export cho phép bạn xuất (export) hình ảnh hoặc asset từ thiết kế.

  • Scale (1x): là tỷ lệ kích thước khi export.
    Web thường dùng 1x hoặc 2x, iOS thường sẽ là 2x hoặc 3x còn Android sẽ có nhiều hơn.
  • Format (PNG): Định dạng file xuất. Có 4 format:
    • PNG: hình ảnh UI, icon
    • JPG: ảnh chụp
    • SVG: icon vector
    • PDF: tài liệu
  • Export: Đây là nút để thực hiện export.
  • Preview: Cho phép xem trước ảnh trước khi export.
    Bạn có thể:
    • phóng to
    • kiểm tra viền
    • xem pixel
    Dùng khi muốn chắc rằng export đúng.
  • Suffix: hậu tố tên file. VD tên file là icon, nếu đặt suffix là @2x thì file export sẽ là icon@2x
    Dùng khi:
    • retina asset
    • nhiều version file
  • Color profile (sRGB): Xác định hệ màu của file export.
    • sRGB: Đây là chuẩn màu phổ biến cho web, mobile, browser. Hầu hết trường hợp nên để sRGB.
    • Display P3: là một color space (không gian màu) rộng hơn sRGB. Điều đó nghĩa là nó có thể hiển thị nhiều màu hơn và rực hơn. Thường dùng cho iOS app, Mac app hoặc thiết kế media cao cấp.
  • Image resampling: Quyết định cách Figma xử lý pixel khi scale ảnh.
    Detailed: giữ chi tiết tốt hơn, ảnh sắc nét
    Basic: phù hợp export nhanh, preview. Ảnh có thể hơi mềm hoặc hơi blur, chi tiết nhỏ có thể mất.
  • Ignore overlapping layers: Nếu bật, Figma bỏ qua các layer bị che khuất hoàn toàn.
