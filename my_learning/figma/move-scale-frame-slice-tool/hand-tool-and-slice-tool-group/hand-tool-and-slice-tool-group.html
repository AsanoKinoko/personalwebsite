2. Nhóm công cụ Hand tool và Slice

  • Frame (F): Tạo ra các trang mẫu để làm việc. Ví dụ iphone, website. Frame = khung thiết kế (artboard). Trong UI/UX, Frame là container quan trọng nhất.
    Frame hỗ trợ:
    • Auto layout
    • Constraints
    • Grid layout
    • Prototype
    • Scroll
  • Section (Shift + S): Section dùng để nhóm nhiều frame lại với nhau để tổ chức file cho dễ nhìn. Nó giống như folder trong canvas.
    • Mục đích của Section
    • Tổ chức project lớn
    • Chia module UI
    • Dễ navigate
    Section không ảnh hưởng layout UI.
  • Slice (S): Slice dùng để chọn một vùng để export (xuất ảnh). Nó giống như cắt một phần thiết kế để xuất file.
    Slice dùng khi:
    • Export asset
    • Xuất hình ảnh
    • Xuất icon
    • Xuất một phần layout
