Hệ thống lưới (Layout Grid System)

Layout Grid là hệ thống lưới giúp bạn căn chỉnh UI một cách chính xác và nhất quán

Nó giúp:

  • Canh thẳng hàng (alignment)
  • Giữ spacing đều
  • Responsive design
  • Làm design giống dev (rất quan trọng)

Grid không phải để "trang trí". Nó là hệ thống logic giúp UI scale + responsive + maintain

Có 3 loại Layout Grid trong Figma:

a. Grid (ô vuông nhỏ)

Giống như giấy kẻ ô

Ví dụ: 10px x 10px

Dùng cho:

  • Pixel-perfect
  • Icon
  • Micro spacing (padding, margin nhỏ)
  • Size: kích thước ô (VD: 8, 10, 4)
  • Color: màu grid
  • Opacity: độ mờ

Best practice: Dùng 8px grid system (chuẩn industry). Cụ thể là tất cả spacing nên theo: 4 / 8 / 16 / 24 / 32 / 48 / 64... Không nên để tùy ý (13px, 17px,...)

b. Columns (cột)

Chia layout thành nhiều cột

Dùng cho:

  • Web design
  • Landing page
  • Dashboard

Ví dụ:

  • 12 columns (chuẩn web)
  • 8 columns (tablet)
  • 4 columns (mobile)
  • Count: số cột (VD: 12)
  • Type (quan trọng)
    • Left: Cột bắt đầu từ trái
    • Right: Bắt đầu từ phải
    • Center: Căn giữa
    • Stretch: Giãn full width

    Web nên dùng Stretch

  • Margin: khoảng cách từ mép frame

    Ví dụ:

    • Desktop: 120px
    • Tablet: 64px
    • Mobile: 16px
  • Gutter: khoảng cách giữa các cột. Thường là 16px / 24px

c. Rows (hàng)

Chia theo chiều dọc frame

Dùng cho:

  • Vertical rhythm
  • Danh sách (list)
  • Table

Các cài đặt tương tự như Column nhưng theo chiều dọc.

Những lỗi người mới hay mắc:

  • Không dùng grid → layout lệch
  • Dùng grid nhưng không follow
  • Không consistent margin
  • Mix nhiều hệ grid

Tips nâng cao (Senior level):

  • Toggle grid nhanh: Shift + G
  • Dùng nhiều grid cùng lúc: Grid (8px) + Columns (12)
  • Dùng grid cho design system: Giữ consistency toàn app
  • Grid là cầu nối với dev: Frontend dev sẽ dùng:
    • CSS Grid
    • Flexbox

    Bạn design đúng grid → dev code nhanh hơn rất nhiều
