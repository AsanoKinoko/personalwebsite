Các thành phần chính trong Figma

1.1 File

Một File là toàn bộ project thiết kế.

Ví dụ:

Website Project
 ├ Landing page
 ├ Dashboard
 ├ Design system

1.2 Page

Một file có thể có nhiều Page.

Ví dụ:

Page 1: Wireframe
Page 2: UI Design
Page 3: Prototype

1.3 Frame

Frame = Artboard. Nó đại diện cho:

  • màn hình web
  • màn hình mobile
  • section layout

Ví dụ:

Frame:
- Desktop 1440px
- Mobile 390px

Frame thường dùng để thiết kế layout UI.

1.4 Layers

Mọi thứ trong Figma đều là Layer.

Ví dụ:

Frame
 ├ Rectangle
 ├ Image
 ├ Text
 └ Button

Các loại layer phổ biến:

  • Shape (rectangle, circle…)
  • Text
  • Image
  • Group
  • Component
