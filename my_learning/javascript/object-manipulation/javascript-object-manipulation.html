Object Manipulation - Thao tác với đối tượng

Object (đối tượng) là một tập hợp các cặp key–value (thuộc tính và giá trị), là một trong những kiểu dữ liệu quan trọng nhất trong JavaScript.

Các chủ đề chính

  • Khái niệm cơ bản - Các khái niệm, cách khởi tạo, truy cập và thao tác với object
  • Kiểu key, thứ tự, bình đẳng - Kiểu key, Symbols và các đặc điểm về thứ tự và bình đẳng
  • Kỹ thuật tạo & gộp - Property Shorthand, Computed Property Names, Spread syntax, Destructuring, Optional chaining & Nullish coalescing
  • Duyệt & liệt kê - Các phương thức liệt kê thuộc tính và duyệt object
  • Mô tả thuộc tính - Property Descriptors, Object.defineProperty(), Getter/Setter, Object.defineProperties()
  • Bất biến & kiểm soát biến đổi - Object.preventExtensions(), Object.seal(), Object.freeze(), Deep Freeze
  • Prototype & Kế thừa - Prototype Chain, Object.create(), Object.getPrototypeOf(), class syntax
  • Clone (Sao chép) - Nông vs Sâu - Shallow Clone, Deep Clone với structuredClone(), JSON Clone
  • Serialization & Deserialization - JSON.stringify(), JSON.parse(), replacer và reviver
  • Proxy & Reflect - Proxy wrapper, traps, meta-programming
  • Kiểu hoá (TypeScript - nếu dùng) - Type definitions, Utility types, Readonly, as const, Literal & Union types
  • Tổng kết + Lưu ý quan trọng - Hiệu năng, Map vs Object, API cheat sheet, Lỗi phổ biến
