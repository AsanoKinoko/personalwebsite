5. Kết hợp literal types + union types

VD:

type Mode = 'dark' | 'light';

type Config = { mode: Mode; retry: number };

const cfg: Config = { mode: 'dark', retry: 3};

Lợi ích:

  • Dễ dàng kiểm tra cấu hình hợp lệ.
  • IntelliSense sẽ tự động gợi ý 'dark' | 'light'.

Kết hợp literal types với union types giúp tạo ra các kiểu dữ liệu chặt chẽ và an toàn, đảm bảo chỉ các giá trị cụ thể mới được chấp nhận.
