2. Spread và Destructuring trong TypeScript

Khi dùng spread syntax như {...u, name}, TypeScript sẽ suy luận kiểu của object mới từ object cũ và kiểm tra nếu bạn ghi đè thuộc tính có kiểu không tương thích.

VD:

const u = { id: '1', name: 'Tom' };
const newU = {...u, age: 20 };
// OK: newU có { id, name, age }

TypeScript sẽ tự động suy luận kiểu của newU dựa trên kiểu của u và các thuộc tính mới được thêm vào.
