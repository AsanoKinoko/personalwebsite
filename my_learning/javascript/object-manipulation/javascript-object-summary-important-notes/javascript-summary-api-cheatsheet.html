3. Các API trọng yếu (cheat sheet)

Kiểm tra/duyệt:

  • in - Kiểm tra thuộc tính (bao gồm prototype chain)
  • Object.hasOwn(obj, key) - Kiểm tra thuộc tính riêng
  • Object.keys(obj) - Lấy mảng keys (string, enumerable)
  • Object.values(obj) - Lấy mảng values
  • Object.entries(obj) - Lấy mảng [key, value] pairs
  • Reflect.ownKeys(obj) - Lấy tất cả keys (string + symbol)

Descriptors:

  • Object.getOwnPropertyDescriptor(obj, key) - Lấy descriptor của một thuộc tính
  • Object.getOwnPropertyDescriptors(obj) - Lấy tất cả descriptors
  • Object.defineProperty(obj, key, desc) - Định nghĩa một thuộc tính
  • Object.defineProperties(obj, descriptors) - Định nghĩa nhiều thuộc tính

Trạng thái thay đổi:

  • Object.preventExtensions(obj) - Ngăn thêm thuộc tính mới
  • Object.seal(obj) - Niêm phong (không thêm/xóa, có thể sửa)
  • Object.freeze(obj) - Đóng băng (không thay đổi gì)
  • Object.isExtensible(obj) - Kiểm tra có thể mở rộng
  • Object.isSealed(obj) - Kiểm tra đã niêm phong
  • Object.isFrozen(obj) - Kiểm tra đã đóng băng

Prototype:

  • Object.create(proto) - Tạo object với prototype
  • Object.getPrototypeOf(obj) - Lấy prototype
  • Object.setPrototypeOf(obj, proto) - Đặt prototype (chậm, tránh dùng)

Sao chép & chuyển đổi:

  • Object.assign(target, ...sources) - Gộp/sao chép nông
  • {...obj} - Spread syntax (sao chép nông)
  • Object.fromEntries(entries) - Chuyển entries thành object
  • structuredClone(obj) - Sao chép sâu (ES2021+)

Meta:

  • new Proxy(target, handler) - Proxy wrapper
  • Reflect.* - Các hàm meta-programming (get, set, has, deleteProperty, ownKeys, apply, construct...)

Symbol:

  • Symbol(description) - Tạo symbol mới
  • Object.getOwnPropertySymbols(obj) - Lấy tất cả symbol keys
  • Well-known symbols: Symbol.iterator, Symbol.toStringTag, Symbol.toPrimitive...
