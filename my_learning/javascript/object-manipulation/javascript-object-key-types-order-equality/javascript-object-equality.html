So sánh giá trị (Equality in Object Manipulation)

So sánh object với object:

Object được so sánh theo tham chiếu, không phải theo nội dung.

const a = {x: 1};
const b = {x: 1};
const c = a;
console.log(a === b); // false (khác vùng nhớ)
console.log(a === c); // true (cùng tham chiếu)

So sánh giá trị bằng === (strict equality):

Hoạt động bình thường nhưng có 2 điểm "lạ":

console.log(NaN === NaN); // false
console.log(+0 === -0); // true

So sánh chính xác hơn bằng Object.is():

Method này sửa 2 điểm "không hợp lý" của ===, còn lại hoạt động tương tự strict equality.

console.log(Object.is(NaN, NaN)); // true
console.log(Object.is(+0, -0)); // false
console.log(Object.is(5, 5)); // true
console.log(Object.is({}, {})); // false (khác vùng nhớ)
