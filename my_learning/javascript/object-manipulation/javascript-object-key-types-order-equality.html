Kiểu key, thứ tự, bình đẳng

Object key hợp lệ là String hoặc Symbol. Cụ thể:

  • Nếu bạn dùng số, nó sẽ tự động được ép sang chuỗi.
  • Nếu bạn dùng Symbol, đó là một key đặc biệt, không xung đột với key chuỗi khác.
const sym = Symbol("id");
const obj = {
  1: "one",
  name: "Alice",
  [sym]: "secret",
};

console.log(Object.keys(obj)); // ["1", "name"]
console.log(Object.getOwnPropertySymbols(obj)); // [Symbol(id)]

Lưu ý:

  • 1, "1" và Symbol("1") là 3 key hoàn toàn khác nhau.
  • Key dạng số luôn bị chuyển thành string, chỉ khác là JS đối xử đặc biệt khi liệt kê thứ tự (giải thích ở phần tiếp theo).
