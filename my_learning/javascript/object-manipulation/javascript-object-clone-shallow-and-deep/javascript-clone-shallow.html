1. Clone Nông (Shallow Clone)

Đặc điểm:

  • Chỉ sao chép cấp đầu tiên (level 1).
  • Nếu object lồng nhau → vẫn dùng chung tham chiếu.

Cách phổ biến:

Dùng toán tử spread:

const c1 = { ...o };

Dùng Object.assign:

const c2 = Object.assign({}, o);

VD:

const o = {a: 1, nested: {b: 2 } };
const c = { ...o };
c.nested.b = 99;
console.log(o.nested.b); // 99 (vẫn bị ảnh hưởng)

Điều quan trọng: nested là object bên trong, được sao chép bằng tham chiếu, không tạo bản sao mới. Vì thế gọi là shallow clone (sao chép nông).
