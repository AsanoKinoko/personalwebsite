Built-in Functions

Built-in Function là những hàm được xây dựng sẵn trong JavaScript, chỉ cần lấy ra sử dụng mà không cần định nghĩa lại.

alert()

alert('Thông báo');

Hiển thị hộp thoại thông báo với nút OK.

console

Console chứa nhiều phương thức, để in ra thông báo bên tab Console của Dev tool trình duyệt. Để truy cập các phương thức của console thì chúng ta sử dụng dấu chấm ..

  • console.log() 
    console.log('Thông báo');

    In ra dòng log có nội dung là "Thông báo" bên tab Console. Có thể đưa vào một số hoặc một biến.

    console.log('Hello', 123, myVariable);
  • console.warn() 
    console.warn('Cảnh báo');

    In ra dòng log cảnh báo (có biểu tượng ! màu vàng ở trước).

  • console.error() 
    console.error('Lỗi');

    In ra dòng log báo lỗi (có biểu tượng lỗi màu đỏ ở trước).

confirm()

confirm('Message');

Tạo ra hộp thoại Confirm với hai nút: OK và Cancel. Trả về true nếu người dùng nhấn OK, false nếu nhấn Cancel.

prompt()

prompt('Message');

Tạo ra hộp thoại giống hộp thoại Confirm nhưng có thêm ô input cho người dùng nhập. Trả về giá trị người dùng nhập vào hoặc null nếu hủy.

setTimeout()

setTimeout(function() {
    // Coding here…
}, x);

Cho một đoạn code được chạy sau một khoảng thời gian x, x tính theo millisecond (1 giây = 1000 milliseconds). Ví dụ:

setTimeout(function() {
    console.log('Chạy sau 2 giây');
}, 2000);

setInterval()

setInterval(function() {
    // Coding here…
}, x);

Cho một đoạn code, cứ định kỳ sau một khoảng thời gian x nó lại được chạy một lần, x tính theo millisecond. Ví dụ:

setInterval(function() {
    console.log('Chạy mỗi 1 giây');
}, 1000);
