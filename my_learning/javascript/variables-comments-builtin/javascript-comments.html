Comments (Ghi chú)

Comment một dòng

Sử dụng 2 dấu gạch chéo // trước comment:

// [Nội dung comment]
var name = 'JavaScript'; // Đây là một comment ở cuối dòng

Comment nhiều dòng (khối)

Sử dụng /* để bắt đầu và */ để kết thúc:

/* 
   [Nội dung comment]
   Có thể viết nhiều dòng
   trong khối comment này
*/

/* Comment một dòng với cú pháp khối */

Phím tắt

Trong hầu hết các trình soạn thảo code, bạn có thể sử dụng phím tắt Ctrl + / để comment/uncomment dòng code đang được chọn.

Lưu ý

  • Comments không được thực thi bởi JavaScript engine
  • Comments giúp code dễ đọc và dễ bảo trì hơn
  • Nên viết comments rõ ràng, giải thích tại sao hơn là cái gì
