Toán tử gán (Assignment Operators)

Sử dụng để gán giá trị cho một biến.

Toán tử Tương đương
= x = y
+= x = x + y
-= x = x - y
*= x = x * y
/= x = x / y
**= x = x ** y

Ví dụ

var x = 10;
 x += 5;   // x = x + 5, kết quả: x = 15
 x -= 3;   // x = x - 3, kết quả: x = 12
 x *= 2;   // x = x * 2, kết quả: x = 24
 x /= 4;   // x = x / 4, kết quả: x = 6
 x **= 2;  // x = x ** 2, kết quả: x = 36
