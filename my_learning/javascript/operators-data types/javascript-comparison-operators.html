Toán tử so sánh (Comparison Operators)

Các toán tử so sánh được sử dụng để so sánh hai giá trị và trả về kết quả là true hoặc false.

Toán tử Chức năng
== Bằng
!= Không bằng
> Lớn hơn
< Nhỏ hơn
>= Lớn hơn hoặc bằng
<= Nhỏ hơn hoặc bằng

Ví dụ

var a = 5;
var b = 3;
console.log(a == b);  // false
console.log(a != b);  // true
console.log(a > b);   // true
console.log(a < b);   // false
console.log(a >= b);  // true
console.log(a <= b);  // false

if else

Chúng ta sẽ hay gặp cấu trúc điều kiện if else sử dụng các toán tử so sánh để thực thi code dựa trên điều kiện.

if (conditions/variable) {
    // Coding here...
} else {
    // Coding here...
}

Ví dụ sử dụng if else

var age = 18;

if (age >= 18) {
    console.log('Bạn đã đủ tuổi');
} else {
    console.log('Bạn chưa đủ tuổi');
}

var score = 85;
if (score >= 90) {
    console.log('Xuất sắc');
} else if (score >= 70) {
    console.log('Khá');
} else {
    console.log('Cần cố gắng');
}
