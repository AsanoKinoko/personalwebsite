Toán tử số học (Arithmetic Operators)

Các toán tử số học được sử dụng để thực hiện các phép tính toán học cơ bản.

Toán tử Chức năng
+ Cộng
- Trừ
* Nhân
** Lũy thừa
/ Chia
% Chia lấy số dư
++ Tăng 1 giá trị số
-- Giảm 1 giá trị số

Lưu ý

  • Toán tử ++/-- khi dùng là hậu tố (Postfix) sẽ tăng giá trị của biến lên 1 và trả về giá trị trước khi tăng.
  • Toán tử ++/-- khi dùng là tiền tố (Prefix) sẽ tăng giá trị của biến lên 1 và trả về giá trị sau khi tăng.
  • Toán tử + nếu một trong 2 cái không phải là số thì nó sẽ trở thành toán tử nối chuỗi (String Operators).

Ví dụ

var a = 5; var b = 3;
console.log(a + b);  // 8
console.log(a - b);  // 2
console.log(a * b);  // 15
console.log(a ** b); // 125
console.log(a / b);  // 1.666...
console.log(a % b);  // 2

var x = 5;
console.log(x++);    // 5 (trả về giá trị trước khi tăng)
console.log(x);      // 6

var y = 5;
console.log(++y);    // 6 (trả về giá trị sau khi tăng)
console.log(y);      // 6
