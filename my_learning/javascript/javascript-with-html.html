Sử dụng Javascript với HTML

Internal

Đặt trực tiếp cặp thẻ <script> vào mã HTML và viết javascript giữa cặp thẻ này.

<script>
    alert('Xin chào các bạn!')
</script>

External

Sử dụng thẻ <script> với thuộc tính src để liên kết đến file JavaScript bên ngoài:

<script src="đường_dẫn_tới_file.js"></script>

Trong trường hợp sử dụng file .js thì nội dung của file không được chứa thẻ <script>.
