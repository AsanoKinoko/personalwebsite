So sánh, chuyển kiểu & "weird corners"

So sánh tham chiếu

[] === [] là false. So nội dung cần tự viết (deepEqual), hoặc dùng JSON.stringify khi dữ liệu đơn giản (nhưng coi chừng thứ tự key, kiểu đặc biệt).

Tránh "==" với mảng

Tránh "==" với mảng; dùng "===" vì "==" sẽ ép kiểu với mảng, dễ gây bug.

Ví dụ

const arr1 = [1, 2, 3];
const arr2 = [1, 2, 3];

console.log(arr1 === arr2);  // false - so sánh tham chiếu

// So sánh nội dung với JSON.stringify (chỉ dùng khi dữ liệu đơn giản)
console.log(JSON.stringify(arr1) === JSON.stringify(arr2));  // true

// Lưu ý: JSON.stringify có thể gặp vấn đề với:
// - Key không đảm bảo thứ tự trong object 
// - giá trị undefined, function, Symbol
// - Kiểu Date (bị convert thành chuỗi ISO)
// - Map, Set, BigInt
