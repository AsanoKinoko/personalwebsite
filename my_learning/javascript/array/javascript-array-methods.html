Các method trong mảng (Nhóm theo chức năng)

1. Thêm/xóa/chèn (thường đột biến mảng gốc)

Phương thức Mô tả Đột biến? Ví dụ
push(...items) Thêm phần tử vào cuối mảng, trả về độ dài mới. Có let a=[1,2]; a.push(3,4); // [1,2,3,4]
pop() Bỏ phần tử cuối cùng, trả về phần tử đó. Có let a=[1,2,3]; a.pop(); // trả 3, a=[1,2]
unshift(...items) Thêm phần tử vào đầu mảng, trả về độ dài mới. Có let a=[2,3]; a.unshift(1); // [1,2,3]
shift() Bỏ phần tử đầu tiên, trả về phần tử đó. Có let a=[1,2,3]; a.shift(); // trả 1, a=[2,3]
splice(start, deleteCount, ...items) Xóa hoặc chèn phần tử giữa mảng. Trả về mảng chứa phần tử bị xóa. Có let a=[1,2,3,4]; a.splice(1,2,'x','y'); // a=[1,'x','y',4]
toSpliced(start, deleteCount, ...items) Giống splice, nhưng không đổi mảng gốc, trả mảng mới. Không let a=[1,2,3]; let b=a.toSpliced(1,1,9); // b=[1,9,3], a=[1,2,3]

2. Cắt/gộp/điền/copy nội bộ

Phương thức Mô tả Đột biến? Ví dụ
slice(start?, end?) Cắt đoạn [start, end) tạo mảng mới. Không [1,2,3,4].slice(1,3); // [2,3]
concat(...arrs) Nối nhiều mảng hoặc giá trị tạo ra mảng mới. Không [1,2].concat([3,4]); // [1,2,3,4]
fill(value, start?, end?) Điền giá trị vào đoạn [start, end) Có let a=[1,2,3]; a.fill(0,1,3); // [1,0,0]
copyWithin(target, start, end?) Sao chép đoạn [start, end) đè lên từ vị trí target Có let a=[1,2,3,4,5]; a.copyWithin(0,3); // [4,5,3,4,5]

3. Sắp xếp/đảo

Phương thức Mô tả Đột biến? Ví dụ
reverse() Đảo ngược thứ tự phần tử. Có [1,2,3].reverse(); // [3,2,1]
toReversed() Trả bản đảo của mảng (không đổi mảng gốc). Không let a=[1,2,3]; a.toReversed(); // [3,2,1], a=[1,2,3]
sort(compareFn?) Sắp xếp theo so sánh. Stable sort từ ES2019+. Có [3,1,2].sort((a,b)=>a-b); // [1,2,3]
toSorted(compareFn?) Trả bản sắp xếp mới, mảng cũ giữ nguyên. Không let a=[3,1,2]; a.toSorted(); // [1,2,3]

4. Biến đổi bất biến từng phần tử

Phương thức Mô tả Đột biến? Ví dụ
with(index, value) Trả bản sao của mảng, trong đó phần tử index được thay bởi value. Không đổi mảng gốc. Không let a=[1,2,3]; a.with(1,99); // [1,99,3], a=[1,2,3]

5. Tìm kiếm và truy vấn (Search and Query)

Phương thức Mô tả Trả về Ví dụ
indexOf(value, fromIndex?) Tìm từ đầu, trả index đầu tiên hoặc -1. Index [1,2,3,2].indexOf(2); // 1
lastIndexOf(value) Tìm từ cuối, trả index cuối cùng. Index [1,2,3,2].lastIndexOf(2); // 3
includes(value, fromIndex?) Kiểm tra có phần tử bằng (===). Boolean [1,2,3].includes(2); // true
find(fn) Trả phần tử đầu tiên thỏa mãn điều kiện. Value [1,2,3].find(x=>x>1); // 2
findLast(fn) Từ cuối lên → phần tử đầu tiên thỏa mãn điều kiện. Value [1,2,3].findLast(x=>x>1); // 3
findLastIndex(fn) Từ cuối lên → index đầu tiên thỏa mãn điều kiện. Index [1,2,3].findLastIndex(x=>x>1); // 2
some(fn) Có ít nhất một phần tử thỏa? Boolean [1,3,5].some(x=>x%2===0); // false
every(fn) Tất cả phần tử thỏa? Boolean [2,4,6].every(x=>x%2===0); // true

6. Ánh xạ / Lọc / Gom (Functional)

Phương thức Mô tả Trả về Ví dụ
map(fn) Ánh xạ mỗi phần tử thành giá trị mới. Mảng mới [1,2,3].map(x=>x*2); // [2,4,6]
filter(fn) Giữ lại phần tử thỏa điều kiện. Mảng mới [1,2,3].filter(x=>x>1); // [2,3]
flat(depth=1) "Dẹt" mảng con lồng nhau. Mảng mới [1,[2,[3]]].flat(2); // [1,2,3]
flatMap(fn) Kết hợp map + flat(1). Mảng mới [1,2,3].flatMap(x=>[x,x*2]); // [1,2,2,4,3,6]
reduce(fn, init?) Tích lũy từ trái → phải. Giá trị [1,2,3].reduce((s,x)=>s+x,0); // 6
reduceRight(fn, init?) Tích lũy từ phải → trái. Giá trị ["a","b","c"].reduceRight((s,x)=>s+x,""); // "cba"

7. Chuỗi hóa & Hiển thị (Stringification & Display)

Phương thức Mô tả Trả về Ví dụ
join(sep=',') Nối phần tử thành chuỗi, cách nhau bởi sep. Chuỗi mới [1,2,3].join('-'); // "1-2-3"
toString() Tự động giống join(','). Chuỗi mới [1,2,3].toString(); // "1,2,3"
toLocaleString() Chuỗi hóa theo định dạng vùng. Chuỗi mới [1000].toLocaleString('vi-VN'); // "1.000"

8. Iterator (dùng trong for...of)

Phương thức Mô tả Trả về Ví dụ
keys() Trả iterator các chỉ mục (0,1,2,...). Tạo một iterator chứa chỉ mục const arr = ['a', 'b', 'c']; const keys = arr.keys(); console.log([...keys]); // [0, 1, 2]
values() Trả iterator các giá trị. Trả về iterator chứa các giá trị trong mảng const arr = ['a', 'b', 'c']; const values = arr.values(); console.log([...values]); // ["a", "b", "c"]
entries() Trả iterator [index, value]. Trả về iterator chứa cặp [index, value] cho từng phần tử const arr = ['a', 'b', 'c']; const entries = arr.entries(); console.log([...entries]); // [[0, "a"], [1, "b"], [2, "c"]]

Lưu ý

  • sort() mặc định theo chuỗi UTF-16: ['2','10','1'].sort() → ['1','10','2'].
  • includes(NaN) là true nếu có NaN (khác với indexOf(NaN) luôn -1).
  • forEach không break/return để dừng sớm → dùng for..of hoặc some/find.
  • map giữ nguyên độ dài (kể cả lỗ); filter có thể rút ngắn.
  • Phương thức "toXxx" (toSorted, toReversed, toSpliced) trả mảng mới, không đổi mảng gốc – phù hợp lập trình bất biến.
  • Typed arrays (như Uint8Array) có API giống một phần Array nhưng hạn chế: cố định độ dài, chỉ số kiểu số.
