Tổng quan

Next.js là một framework của React dùng để xây dựng web application full-stack.

Nó hỗ trợ:

  • Server-Side Rendering (SSR)
  • Static Site Generation (SSG)
  • API routes (backend trong cùng project)
  • Routing tự động theo file
  • Tối ưu SEO
  • Tối ưu hiệu năng
